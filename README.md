NutriAlly is a sophisticated, mobile-optimized web application designed to create personalized Indian diet plans and gym supplement stacks. It leverages the power of Google's Gemini AI to provide culturally relevant nutrition advice.

## ‚ú® Features

- **Personalized Metabolic Profiling**: Calculates BMI, BMR, TDEE, and target calories.
- **Indian Context Meals**: 7-day meal plans featuring regional Indian cuisines.
- **Smart Supplement Stack**: Tailored recommendations (Whey, Creatine, Ashwagandha, etc.).
- **PWA (Progressive Web App)**: Fully installable on **Android and iOS**.
- **Cross-Device Ready**: Works on desktops, iPhones, and Android phones.

## üöÄ One-Click "Install" (PWA)

To "install" this app on your device:
- **Android**: Open the link in Chrome, wait for the **"Install App"** button at the bottom, or select it from the browser menu.
- **iOS**: Open in Safari, tap the **Share** button (box with upward arrow), and select **"Add to Home Screen"**.
